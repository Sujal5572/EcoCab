# 🚀 Mobility Platform

A scalable backend system for a smart demand-based ride-sharing platform, designed to optimize ride allocation during peak hours by enabling dynamic ride sharing along routes.

---

## 📌 Features

- 🚗 Ride booking system
- 📍 Location-based matching (route-aware ride sharing)
- 🔄 Real-time communication using WebSockets
- 🔐 Secure APIs with Spring Security
- 📊 Resilient services using Resilience4j
- 🗄️ PostgreSQL + JPA for persistence
- 🧭 Spatial support (PostGIS via Hibernate Spatial)
- 🔁 Database versioning using Flyway

---

## 🛠️ Tech Stack

- Backend: Spring Boot (Java 17)
- Database: PostgreSQL
- ORM: Spring Data JPA + Hibernate Spatial
- Security: Spring Security
- Realtime: WebSockets
- Migration: Flyway
- Build Tool: Maven
